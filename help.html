TERRY BOT

Welcome to the Terry documentation. All commands will be documented here.

Commands:

/help: Get help

/kill: Quits Terry

/gh: Open TerryBot github

/gs (query): Google Search the query.

/ops (site): Open the site quickly

/bgc (filename): Open a file to set as the background. NOTE: Image must be named exactly (without any folders, just the name) and put in the Images folder.

/status: Check TerryBot's status

/ms (query): (Music) Search for a song on Spotify

/yt (query): Search for a Youtube video.

/fontcolor (hex code or color name): Set a specific font color to your liking

/reset: Resets Terry's memory. Run twice to confirm.

/mode: Switches Terry out of a chat bot mode or a search mode depending on what you need him to do.
